# **App Name**: StudyAI

## Core Features:

- Question Answering: Allows students to ask questions related to their studies and receive answers generated by AI.
- "Explain Like I'm a Beginner" Mode: Simplifies complex topics and explanations into easy-to-understand language using AI.
- Topic Summarization: Generates short summaries of topics using AI.
- Quiz Generation: Generates quizzes from a given topic using AI, which analyzes the topic, creates multiple choice questions, and ranks the questions by difficulty.
- Flashcard Generation: Generates flashcards from a given topic using AI, extracting key terms and definitions, formatting them appropriately for study.
- Simple and Clean UI: Presents a beginner-friendly and clean user interface for ease of navigation and use.
- API integration: Tool to forward all queries to the ChatGPT API and displaying it to the user

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calming, academic feel.
- Background color: Light, off-white (#F8F8FF), for clean reading.
- Accent color: Muted orange (#E59866) to highlight key features.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and accessible look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, clear icons related to studying and learning, such as books, lightbulbs, and magnifying glasses.
- Minimalist layout with a focus on readability and ease of use.
